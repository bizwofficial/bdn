Bridgman Developer Network

概述

欢迎来到Bridgman Developer Network！这是一个由Bridgman创建的代码与文档共享平台。 要开始使用，请点击页面左上角的 图标打开侧边栏（在屏幕较窄的设备或移动设备上可能已经显示），并选择您感兴趣的主题。

仓库索引

BDN包括一些实践和库文档。部分主题（含计划中的项目）如下，您可以据此浏览对应的仓库/说明：

# 主题 参考地址
1 SheeSV https://github.com/bizwofficial/sheesv
2 PlKiller Windows https://github.com/bizwofficial/plkiller-windows
3 Novel In Image https://github.com/3150601355/Novel_In_Image

离线版本

BDN全站提供离线版本。站内跳转链接均使用相对链接，您可以下载并在您的设备或服务器上使用。 请注意：当未部署在服务器时，部分浏览器可能无法正常跳转。 而且如果想要实现完全离线，您还需要从MDUI官网下载离线版本的JS/CSS文件。
